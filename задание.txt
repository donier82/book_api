задание
создайте апи для модели Book, включая слдующие поля:
title(заголовок) обьязателное поле.
author(автор) обьязателное поле.
created_at(дата создание) автоматически создается.
pages(страницы в книге)обьязателное поле указывает количество страниц.
logo(логотип) не обьязателное поле используется для загрузки обложки книги.

Реализуйте функциональность API для работы с книгами:
ModelBook с указанными полями, чтобы возвращались в JSON формате.
BookSerializer для преобразования данных модели Book в JSON.
Viewset mixin  для стандартных CRUD операций.
DefaultRouter для автоматической генерации URL.
Пагинация: на одной книге должно отображаться минимум 10 книг
